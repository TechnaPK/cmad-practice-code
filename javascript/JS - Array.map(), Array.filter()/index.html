Array.forEach() - Array.map() - Array.filter()

Array.forEach()

The forEach() method calls a function (a callback function) once for each array element.

The callback function is invoked with three arguments:

  • the value of the element
  • the index of the element
  • the Array object being traversed

Note: The forEach() method does not change the original array. It returns undefined.

Example

The following code traverses an array and prints its elements:

            
                var fruits = ["Banana", "Orange", "Apple", "Mango"];
                fruits.forEach(function(element, index, array) {
                    console.log(element, index, array);
                });

Array.map()

The map() method creates a new array with the results of calling a function for every array element.

The map() method calls the provided function once for each element in an array, in order.

Note: map() does not change the original array.

Example

The following code traverses an array and prints its elements:

            
                var fruits = ["Banana", "Orange", "Apple", "Mango"];
                var new_fruits = fruits.map(function(element, index, array) {
                    return element + " is at index " + index;
                });
                console.log(new_fruits);

Array.filter()

The filter() method creates an array filled with all array elements that pass a test (provided as a function).

Note: filter() does not change the original array.

Example

The following code traverses an array and prints its elements:

            
                var fruits = ["Banana", "Orange", "Apple", "Mango"];
                var new_fruits = fruits.filter(function(element, index, array) {
                    return element.length > 5;
                });
                console.log(new_fruits);

Array.find()

The find() method returns the value of the first element in the provided array that satisfies the provided testing function.

Note: find() does not change the original array.

Example

The following code finds the first element in an array that satisfies a condition:

            
                var numbers = [4, 9, 16, 25];
                var found = numbers.find(function(element, index, array) {
                    return element > 10;
                });
                console.log(found);
