Javascript Async/Await

"async and await make promises easier to write"

async is a function that returns a promise.
await is used to wait for the promise to resolve.

Async

            
                async function myFunction() {
                    return "Hello"; // it will work as a promise resolved

                    // throw new Error("Something went wrong"); // it will work as a promise rejected
                }

                myFunction()
                  .then(function(value) { console.log(value) })
                  .catch(function(error) { console.log(error) });

Await Syntax

let value = await promise;

The await keyword can only be used inside an async function.

            
                async function getData() {
                    let response = await fetch("https://jsonplaceholder.typicode.com/todos/1");
                    let data = await response.json();

                    console.log(data)
                }
