ES6 Destructuring, ES6 Spread Operator

ES6 Destructuring

ES6 introduces the new syntax for destructuring arrays and objects. It is called destructuring.

Array Destructuring

The syntax is:

            
                const vehicles = ['mustang', 'f-150', 'honda-70'];
                const [car, truck, bike] = vehicles;

The above code is equivalent to:

            
                const car = vehicles[0];
                const truck = vehicles[1];
                const bike = vehicles[2];

Object Destructuring

The following code uses the destructuring syntax to assign the values of the properties of the object to the variables name, age, and city:

            
                const person = {
                    name: 'John',
                    age: 30,
                    city: 'New York'
                };
                const { name, age, city } = person;

The above code is equivalent to:

            
                const name = person.name;
                const age = person.age;
                const city = person.city;

ES6 Spread Operator

The spread operator can be used to create a new array.

            
                const vehicles = ['mustang', 'f-150', 'honda-70', 'honda-100'];
                const newVehicles = [...vehicles];

The above code is equivalent to:

            
                const newVehicles = vehicles.slice();

The spread operator can be used to combine two arrays into a new array easily. For example

            
                const numbersOne = [1, 2, 3];
                const numbersTwo = [4, 5, 6];
                const numbersCombined = [...numbersOne, ...numbersTwo];

The above code is equivalent to:

            
                const numbersCombined = numbersOne.concat(numbersTwo);

The spread operator is used to expand the elements of an array into multiple arguments.

The syntax is:

            
                const vehicles = ['mustang', 'f-150', 'honda-70', 'honda-100'];
                const [car, truck, ...others] = vehicles;

The above code is equivalent to:

            
                const car = vehicles[0];
                const truck = vehicles[1];
                const others = vehicles.slice(2);
