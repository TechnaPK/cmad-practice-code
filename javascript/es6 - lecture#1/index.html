Arrow Functions

This topic is defined very well and in a simple way at:
https://www.w3schools.com/js/js_arrow_function.asp

// Arrow Functions
const add = (a, b) => a + b;
const subtract = (a, b) => a - b;
const multiply = (a, b) => a * b;
const divide = (a, b) => a / b;
const modulus = (a, b) => a % b;
// add(1, 2) = 3
// subtract(1, 2) = -1
// multiply(1, 2) = 2
// divide(1, 2) = 0.5
// modulus(1, 2) = 1

Default Parameters

This topic is defined very well and in a simple way at:

// Default Parameters
const add = (a = 1, b = 1) => a + b;
const subtract = (a = 1, b = 1) => a - b;
const multiply = (a = 1, b = 1) => a * b;
const divide = (a = 1, b = 1) => a / b;
const modulus = (a = 1, b = 1) => a % b;
// add(2, 2) = 4
// subtract(2, 2) = 0
// multiply(2, 2) = 4
// divide(2, 2) = 1
// modulus(2, 2) = 0

The For Of Loop

This topic is defined very well and in a simple way at:
https://www.w3schools.com/js/js_loop_forof.asp

// The For Of Loop
const array = [1, 2, 3, 4, 5];
for (let element of array) {
    console.log(element);
}
// 1
// 2
// 3
// 4
// 5

The For In Loop

This topic is defined very well and in a simple way at:
https://www.w3schools.com/js/js_loop_forin.asp

// The For In Loop
const object = {
    name: 'John',
    age: 30,
    city: 'New York'
};
for (let key in object) {
    console.log(key);
}
// name
// age
// city
